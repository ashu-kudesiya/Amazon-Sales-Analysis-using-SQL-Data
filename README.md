# Amazon Sales Analysis

This project offers a comprehensive analysis of Amazon sales data through interactive visualizations and insightful metrics. It explores sales trends, customer behaviors, and product performance, leveraging a variety of advanced data visualization techniques.

## Overview

- *Sales Performance*: Analyzing revenue trends, high-performing products, and time-based sales distributions.
- *Customer Insights*: Understanding purchase behavior and category preferences.
- *Visualization Techniques*: Utilizing diverse charts and maps for in-depth analysis.

## Requirements

- *Python Notebook (.ipynb)*: Contains the analysis and visualization scripts.
- *Dataset (SQL Dataset)*: Amazon sales data used for the analysis.
- *Libraries Required*:
  - MySQL connector
  - pandas
  - matplotlib
  - seaborn
  - numpy
  - plotly

## Steps for Analysis

### Part 1: Database Connection and Data Loading

- Establish Connection:
    - Connected to a MySQL database (amazon_sales_analysis).
    - Retrieved data from the sales_data table.

- Load Data into DataFrame:
    - Transferred query results into a Pandas DataFrame for analysis.

### Part 2: *Data Cleaning and Transformation*:
- Addressed missing or inconsistent values.
- Standardized column formats for analysis.

### Part 3: Visualization and Insights
To gain meaningful insights from data, various visualization techniques can be employed. These visualizations help in identifying patterns, trends, and relationships, providing a clear and effective way to interpret and communicate the data.

Here are some key visualizations:
  - Heatmap
  - Column Chart
  - Line Chart
  - Pie Chart
  - Scatter Plot
  - Area Chart
  - Clustered Column Chart
  - Dual-Axis Line Chart
  - Bubble Map
  - Bar Chart

---

## Visualizations

| **CHART TYPE**             | **DESCRIPTION**                                                              |
|----------------------------|------------------------------------------------------------------------------|
| **Heatmap**                | Showcased correlations between key variables.                                |
| **Column Chart**           | Displayed sales performance by categories.                                   |
| **Line Trend Chart**       | Tracked monthly or yearly sales trends.                                      |
| **Pie Chart**              | Illustrated the proportional distribution of sales by category.              |
| **Scatter Plot**           | Highlighted relationships between sales and quantity.                        |
| **Area Chart**             | Visualized cumulative sales over time.                                       |
| **Clustered Column Chart** | Compared sales across categories and subcategories.                          |
| **Double Line Chart**      | Showed comparisons of two metrics over time (e.g., revenue vs. quantity).    |
| **Bubble Map Chart**       | Represented geographical sales performance.                                  |
| **Bar Chart**              | Focused on horizontal data distribution.                                     |



### Key Features

- *Interactivity*: Hover and zoom on charts for detailed insights (if interactive charts are implemented).
- *Comparative Analysis*: Enables side-by-side comparisons of metrics.
- *Geographic Analysis*: Visualizes regional sales using bubble maps.
